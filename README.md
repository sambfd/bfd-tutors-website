# BFD Tutors Website

Static one-page website built from the three BFD Tutors audit/presentation documents.

## What is included

- `index.html` - the website content
- `styles.css` - the visual design
- `script.js` - the mobile navigation behaviour
- `assets/sam-finnegan-dehn.png` - portrait image extracted from the supplied presentation
- `robots.txt` - allows search engines to crawl the website
- `sitemap.xml` - tells search engines the canonical public page URL
- `CNAME` - sets `www.bfdtutors.com` as the custom domain for GitHub Pages

## Free hosting options

This folder can be hosted for free on GitHub Pages, Netlify, Cloudflare Pages, or Vercel because it uses only static files.

Recommended route: GitHub Pages or Cloudflare Pages, then point `www.bfdtutors.com` at the host from your domain/DNS settings.

## Search indexing

Once the site is live at `https://www.bfdtutors.com/`, submit `https://www.bfdtutors.com/sitemap.xml` in Google Search Console.
